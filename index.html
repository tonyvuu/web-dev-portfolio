Portfolio

The thrill of being able to create projects and bringing any idea to life is the reason why I love doing this. Here are some current and finish projects. Don't worry! I will continue to add more. 😄

SpotiMix

A React web application created with the goal for users to search any tracks, albums, or top 10 tracks of an artist. I also implemented feature such as dark & light theme mode with some more upcoming feature. It will also have upcoming features such as a payment method with included technologies used like bootstrap and API. (working progress).

Red Stop Watch

A simple stopwatch application built using React.js. The stopwatch allows users to start, stop, and reset the timer. It also displays the elapsed time in hours, minutes, seconds, and milliseconds.

Movie.tv

A React.js web application that allows users to fetch comprehensive information about any movie using a movie API. In addition to basic movie data like title, poster, and release date, this application fetches extended details such as runtime, plot, director, and more using the same API key with a different endpoint.

Studdy Buddy

A full-stack web application created with the goal for users to study their flashcards in a distraction-free environment. Flip between questions and answers to test their knowledge. I also implemented a mini Jeopardy-style game mode with categories and point values. Users can select questions from different categories and earn points.

TaskQuest

This is a To-Do List application built using React, a popular JavaScript library for building user interfaces. The app allows users to create, edit, delete, and mark tasks as done.

About Me

Hi, my name is Tony, and I am a highly ambitious, self-motivated, and tenacious software engineer based in Atlanta, GA. After graduating from Georgia State University in 2022 with a BS in finance, I started to realize that I pursued it only because it was a comfortable, safe choice. I had tested out different majors, but I always ended up going back to finance. Around the time of the pandemic, I began immersing myself in technology. Before I knew it, I was spending hours, then weeks, on anything related to technology. I never had the opportunity to try software development because I felt like I had spent too long in college and should just go straight into the workforce. However, I was luckily introduced to a web development bootcamp held in Atlanta, and I immediately sent in my application and was accepted. I am currently in the third month of the program, which is sponsored by the City of Refuge and partnered with DigitalCrafts to train emerging technologists, including myself, to break into tech. What I have learned from this program is that my true passion lies in coding. Being able to bring your ideas to life and demonstrate them is fascinating. I always look forward to creating and solving problems with the abilities I've learned, and I will continue to improve in every way I can. One thing for sure about this field is that I love the constant challenge and continuous learning. Outside of programming, I have a wide range of hobbies and passions that keep me busy. I grew up playing sports so I am a huge sportsfan (my favorite athlete is Lebron James, yes I know -- basic but he's the GOAT). My favorite sports are basketball and volleyball. I love all kind of food so I eat almost anything you put in front of me (unfortunately). I am very open-minded and be willing to try anything outside of my comfort zone.
